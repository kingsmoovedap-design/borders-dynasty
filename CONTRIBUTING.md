# Contributing to Borders Dynasty

Welcome to the Borders Dynasty project! We appreciate your interest in contributing to our royal blockchain infrastructure.

## How to Contribute

1. **Fork the Repository**: Create your own copy of the project.
2. **Create a Feature Branch**: Work on your changes in a dedicated branch (`git checkout -b feature/amazing-feature`).
3. **Commit Your Changes**: Ensure your commits follow the dynasty's conventions.
4. **Push to the Branch**: Upload your changes to GitHub.
5. **Open a Pull Request**: Submit your changes for royal review.

## Code of Conduct

Please be respectful and professional in all interactions. Our goal is to build a secure and elegant sovereignty platform for everyone.

## Development Setup

- Run `npm install` to set up dependencies.
- Use `npm run dev` to start the development environment.
- Follow the guidelines in `replit.md` for project-specific architecture.
