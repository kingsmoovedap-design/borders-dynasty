Borders Dynasty – Web3 DeFi & Logistics Operating System

Borders Dynasty is a Web3 DeFi sovereign blockchain platform for the CodexChain / Borders Dynasty ecosystem. It unifies an upgradeable ERC20 Borders Sovereign Coin (BSC), on‑chain document verification, and a full AI‑powered logistics OS with multi‑view loadboard, treasury, credit, compliance, and Codex Ecclesia governance.

Upgradeable ERC20 • OpenZeppelin Hardhat • Web3 • MetaMask QFS‑Compatible • ISO‑20022 Ready
Web3 DeFi Core

Borders Sovereign Coin (BSC)

  • • Upgradeable ERC20 (1M cap, roles, pausable, blacklist)

Architecture

Monorepo • APIs • Persistence

Borders Dynasty is a monorepo with microservice‑style packages, API‑first design, and PostgreSQL‑backed persistence.

  • Monorepo: shared logic across apps and packages
  • Backend: Express Logistics API (apps/api/)
  • Codex: Ecclesia hash‑chain service (packages/codex/)
  • Database: PostgreSQL + Drizzle ORM + pg Pool
  • Caching: in‑memory Maps + db‑persistence modules
  • Tooling: Hardhat • Webpack • Babel • OpenZeppelin Upgrades

Security & Compliance: focus on role‑based access, rate limiting, anomaly detection, and governance‑aligned operations.
